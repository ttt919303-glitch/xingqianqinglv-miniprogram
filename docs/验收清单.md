# 行前晴旅验收清单

## 小程序功能

- 首页可显示示例行程、倒计时、准备进度和旅行提示。
- 底部导航包含首页、路线、清单、账单、我的五个入口。
- 旅行详情页可聚合行程路线、交通卡片、旅行备忘、账单预算、地点收藏和行李清单入口，模块卡片可点击跳转。
- 物品清单可按分类显示总数、已准备数量和进度。
- 分类详情可切换全部、未打包、已打包，并支持勾选状态保存。
- 新增页可新增自定义物品，保存后能回到对应分类查看。
- 新增页可新增自定义行程，包含城市、日期、交通、交通时间、耗时和景点安排。
- 行程计划页可展示当天景点、推荐顺序、分段交通建议和预计用时。
- 路线策略可在时间优先、少换乘、距离优先之间切换。
- 路线页支持编辑模式，可上移/下移景点顺序，并自动重新估算结束时间、交通时间、提醒和地图预览。
- 点击获取腾讯路线时，可优先请求腾讯位置服务；接口失败时回退本地路线并显示提示。
- 交通卡片可新增、编辑、删除去程、返程等结构化交通信息。
- 账单页可新增、编辑、删除预算和实际花费，并统计分类金额和超支提醒。
- 地点收藏夹可筛选、新增、编辑、删除收藏地点，并把收藏地点一键加入当前行程。
- 模板页可套用旅行模板，并避免重复添加同名物品。
- 我的页面可查看作品说明、统计信息，并可清空勾选或恢复初始数据。

## 项目材料

- 页面数量为 10 页，符合 4 至 10 页要求。
- 页面深度控制在 2 至 3 层之间。
- 主题积极健康，聚焦旅行出行准备。
- 项目文件命名规范，目录结构清晰。
- 已提供项目书 `行前晴旅微信小程序项目书.md`。
- 已提供 README、自动化说明和本地验证脚本。

## 验证命令

```powershell
npm run verify
```

验证范围包括新增物品、新增行程、路线排序、地图预览、交通卡片、账单编辑删除、地点收藏管理、旅行详情、腾讯地图请求构造、数据重置、JSON 配置、页面文件、tabBar 图标和 JS/CJS 语法。
